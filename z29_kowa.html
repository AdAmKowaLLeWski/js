Pytania teoretyczne z programowania

1. W jaki sposób można określić właściwość danego obiektu?

Właściwość danego obiektu można określić na dwa sposoby:

Sposób 1: Używając zapisu postaci nazwa_obiektu.nazwa_właściwości = wartość;

Przykłady:
auto.marka = "Opel Omega";
auto.rok = 1996;

Sposób 2: Używając zapisu tablicowego nazwa_obiektu["nazwa_właściwości"] = wartość;

Przykład:
auto["cena"] = 25000;

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

Do właściwości i metod można odwołać się używając notacji kropkowej lub tablicowej.

Przykład:
console.log(samochod1.kolor); (wyświetla kolor obiektu samochod1)

3. Wymień cechy charakterystyczne konstruktora.

- Jest wywoływany automatycznie przy tworzeniu obiektu.
- Nosi taką samą nazwę jak klasa.
- Nie zwraca żadnej wartości.
- Może przyjmować argumenty.

4. Konstruowanie obiektu na podstawie przykładu

Obiekt z trzema polami i jedną metodą:

var osoba_1 = {
    nazwisko: "Celmer",
    imie: "Łukasz",
    zawod: "piekarz",
    wyswietl: function () {
        document.write(this.nazwisko + " " + this.imie);
    }
};

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

this odnosi się do bieżącego obiektu. Pozwala na odwołanie się do pól i metod obiektu.

Pola i metody w obiekcie oddzielamy przecinkami.

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

function Klient(nazwisko, imie, zawod) {
    this.nazwisko = nazwisko;
    this.imie = imie;
    this.zawod = zawod;
    this.wypisz = function () {
        alert(this.nazwisko + ' ' + this.imie);
    };
}

7. Znaczenie słowa kluczowego new wraz z przykładem.

Słowo kluczowe new służy do tworzenia nowych obiektów.

Przykład:
var osoba1 = new Klient('Kowalski', 'Jan', 'kierowca');

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

prototype pozwala dodawać nowe metody i właściwości do istniejących obiektów.

function Klient() {
    this.nazwisko = "Celmer";
    this.imie = "Łukasz";
}
Klient.prototype.pisz_dane = function () {
    document.write(this.nazwisko + ' ' + this.imie);
};
Klient.prototype.zawod = 'kierowca';
var osoba1 = new Klient();
osoba1.pisz_dane();

Teraz każdy nowy obiekt utworzony na podstawie Klient będzie miał dodatkowe właściwości i metody.
